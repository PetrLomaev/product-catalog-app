# Project "Product catalog"

## Project Description

Displays a page with a product catalog

The following usage functions are available:

- Contains a main page with a list of products, a display of product categories, filter buttons, and a shopping cart

- Functionality for adding and removing items from the cart

- Functionality for filtering by product categories and sorting by price


## Technologies used

- React, React Hooks

- TypeScript

- Next.js

- Styled-Components


## Minimum System Requirements

Node.js Command Line, npm


## Install 

`git clone git@github.com:PetrLomaev/product-catalog-app.git`

`cd product-catalog-app`

`npm install`

`npm run dev`

Open your browser and go to: http://localhost:3000
